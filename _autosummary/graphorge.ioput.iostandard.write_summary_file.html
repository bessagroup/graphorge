Parameters:

  • summary_directory (str) – Directory where the summary file is written.

  • filename (str, default='summary') – Summary file name.

  • summary_title (str, default=None) – Header of summary file.

  • kwargs (dict) – Keyword-based parameters to be written in summary file.
